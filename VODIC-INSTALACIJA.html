01 — Struktura projekta

Struktura datoteka

Svi ovi fajlovi trebaju biti u korijenskom direktoriju tvoje domene:

neutralizam.eu/
├── index.html ← glavna stranica
├── style.css ← svi stilovi
├── script.js ← Firebase + logika
├── manifest.json ← PWA manifest
├── sw.js ← Service Worker (PWA offline)
├── offline.html ← offline fallback stranica
├── robots.txt ← SEO crawling pravila
├── sitemap.xml ← SEO sitemap
├── nadzor-ks2025.html ← admin panel
└── assets/
├── logo.png ← ikona (min. 512×512px)
├── og-image.png ← Open Graph slika (1200×630px)
├── screenshot-wide.png ← PWA screenshot desktop (1280×720)
├── screenshot-mobile.png ← PWA screenshot mobile (390×844)
└── knjiga.docx ← knjiga za preuzimanje
⚠ Zelenom bojom označene datoteke trebaš kreirati/pribaviti ručno. og-image.png je slika 1200×630px koja se prikazuje pri dijeljenju linka (npr. na WhatsAppu). Može biti screenshot naslovnice ili custom grafika.
02 — Preporučeni hosting

Deployment na .eu domenu — Cloudflare Pages

Cloudflare Pages je besplatan, izuzetno brz (edge CDN) i radi savršeno s Firebase-om. Idealan za statičke stranice kao što je Neutralizam.

Korak po korak

1

Stvori GitHub repozitorij

Idi na github.com/new → stvori novi privatni repozitorij naziva neutralizam. Upload svih datoteka.

2

Cloudflare Pages — poveži repozitorij

Idi na pages.cloudflare.com → Create a project → Connect to Git → odaberi GitHub repo neutralizam.

3

Build postavke

Sve ostavi prazno — nema build commanda ni output direktorija (čisti statički HTML). Klikni Save and Deploy.

4

Povežu .eu domenu

U Cloudflare Pages → Custom domains → unesi neutralizam.eu. Cloudflare automatski postavi DNS i HTTPS.

5

Kupi .eu domenu

Preporučujem direktno kroz Cloudflare Registrar — nema markup-a, samo stvarna cijena. Alternativno: Namecheap ili Google Domains. Prosječna cijena: ~8–12€/god.

6

Automatski deploy pri svakom push-u

Nakon što promijeniš datoteku i push-aš na GitHub, Cloudflare Pages automatski redeploya u roku ~30 sekundi.

✓ Cloudflare Pages automatski dodaje HTTPS, HTTP/2, globalnu CDN mrežu i GZIP kompresiju. Sve besplatno, bez limita posjeta.
03 — Alternativa

GitHub Pages + Cloudflare DNS

Ako već imaš .eu domenu negdje drugdje:

1

Uključi GitHub Pages

Repozitorij → Settings → Pages → Branch: main → / (root) → Save.

2

Dodaj CNAME datoteku

U korijenskom direktoriju repozitorija napravi datoteku CNAME (bez ekstenzije) s jednim retkom:

neutralizam.eu
3

Cloudflare DNS zapisi

U Cloudflare DNS dodaj:

# Tip Ime Vrijednost
A @ 185.199.108.153
A @ 185.199.109.153
A @ 185.199.110.153
A @ 185.199.111.153
CNAME www tvoj-username.github.io
04 — Firebase

Firebase konfiguracija

Autorizirane domene

Firebase blokira prijavu s nepoznatih domena. Moraš dodati svoju .eu domenu u whitelist:

1

Firebase Console

Idi na console.firebase.google.com → projekt neutralizam

2

Authentication → Settings → Authorized domains

Klikni Add domain i dodaj: neutralizam.eu i www.neutralizam.eu

3

Firestore pravila su već postavljena

Tvoja Firebase Security Rules (koje si poslao) su već ispravno konfigurirane za produkciju.

Firestore indeksi (važno za pitanja)

Sortiranje pitanja po glasovima (orderBy("votes", "desc")) zahtijeva kompozitni indeks. Firebase će automatski prikazati link za kreiranje u konzoli (error poruka) pri prvom pokretanju. Alternativno, ručno:

# Firebase Console → Firestore → Indexes → Add index
Collection: questions
Fields:
votes Descending
timestamp Descending
05 — PWA

PWA — Kako instalirati na uređaj

Stranica je potpuna PWA aplikacija. Korisnici je mogu instalirati bez App Storea, direktno s preglednika.

📱 Android (Chrome)

Otvori neutralizam.eu u Chromeu → tapni ⋮ (izbornik) → "Dodaj na početni zaslon" ili se pojavi automatski banner "Instaliraj aplikaciju".

🍎 iPhone / iPad (Safari)

Otvori neutralizam.eu u Safariju → tapni gumb Dijeli (↑) → "Dodaj na početni zaslon". Potvrdi tapom na "Dodaj".

🖥️ Desktop Chrome / Edge

U adresnoj traci klikni ikonu ⊕ (desna strana) ili izbornik → "Instaliraj Neutralizam". Pojavljuje se kao zasebna aplikacija.

🪟 Windows (Edge)

Edge automatski prepoznaje PWA → desno od adresne trake pojavi se ikona aplikacije → klikni → "Instaliraj". Dodaje se u Start menu.

ℹ Za potpunu PWA instalabilnost trebaju postojati slike assets/logo.png (min. 512×512px), assets/screenshot-wide.png (1280×720) i assets/screenshot-mobile.png (390×844). Bez screenshotova PWA i dalje radi, ali nema preview u instalacijskom dijalogu.

Provjera PWA kvalitete

Nakon deploya otvori Chrome DevTools → Lighthouse → pokretni audit. Ciljaj PWA score 100, SEO 100, Performance 90+.

⚠ Service Worker radi samo na HTTPS domeni, ne na localhost (osim s --allow-insecure-localhost flagom). Nakon deploya na Cloudflare Pages HTTPS je automatski.
06 — SEO

SEO & Open Graph — Što još treba napraviti

og-image.png — najvažnija stvar

Ova slika se prikazuje pri dijeljenju linka na Facebooku, WhatsAppu, Discordu, iMessageu… Dimenzije: 1200×630px.

Besplatni alati za kreiranje: Canva (predlošci za "Open Graph" / "Facebook Cover") ili ogimage.org.

Spremi kao assets/og-image.png i stavi na server.

Google Search Console

1

Dodaj stranicu

Idi na search.google.com/search-console → Add property → unesi https://neutralizam.eu/

2

Verifikacija putem DNS-a

Google daje TXT zapis koji dodaješ u Cloudflare DNS → verifikacija je automatska u roku par minuta.

3

Pošalji sitemap

Search Console → Sitemaps → unesi https://neutralizam.eu/sitemap.xml → Submit.

Facebook / Meta Sharing Debugger

Nakon što postaviš og-image.png, provjeri prikaz linka na: developers.facebook.com/tools/debug — unesi URL i klikni "Scrape Again" za osvježavanje cachea.

07 — Checklist

Finalni deployment checklist

  • ✓ index.html — SEO meta tagovi, Open Graph, Structured Data, PWA linkovi

  • ✓ script.js — Firebase, SW registracija, sve značajke
  • ✓ manifest.json — icons, shortcuts, screenshots, kategorije
  • ✓ sw.js — Service Worker v1.2, stale-while-revalidate, offline fallback
  • ✓ offline.html — N-tematski offline ekran
  • ✓ robots.txt — admin panel blokiran, sitemap naveden
  • ✓ sitemap.xml — sve javne stranice navedene
  • ✓ Firebase Security Rules — postavljene i testirane
  • ✓ Firebase Authorized Domains — neutralizam.eu dodan
  • ○ assets/og-image.png — kreirati ručno (1200×630px)
  • ○ assets/screenshot-wide.png i screenshot-mobile.png — kreirati ručno
  • ○ Google Search Console — dodaj stranicu i pošalji sitemap
  • ○ Kupi domenu neutralizam.eu i povežu s Cloudflare Pages
  • ○ Firestore kompozitni indeks za questions kolekciju (votes desc)
✓ Sve datoteke su produkcijski spremnne. Jedino što moraš ručno napraviti je kupnja domene, kreiranje OG slike i Google Search Console verifikacija.
